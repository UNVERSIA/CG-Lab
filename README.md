# CG-Lab
计算机图形学课程实验一：万有引力粒子群

项目简介：
项目基于 Taichi 语言实现，利用 GPU 并行计算能力，创建了一个可与鼠标交互的粒子引力系统。
粒子会被鼠标位置吸引、移动、反弹，整体效果流畅、视觉柔和。

## 一、项目架构
```plaintext
CG-Lab/
└── src/
    └── Work0/
        ├── __init__.py      # 模块标识文件 (空文件，标记Work0为Python包)
        ├── config.py        # 全局参数配置中心 (所有可调节参数集中管理)
        ├── physics.py       # GPU并行物理计算核心 (粒子运动/交互逻辑)
        └── main.py          # 程序入口+界面渲染 (窗口/鼠标交互/粒子绘制)
```
## 二、代码逻辑
1. 初始化
   - 在 config.py 中定义粒子数量、颜色、引力、阻力等全局参数。
   - 在 physics.py 中通过 Taichi 开辟显存空间，存储粒子位置、速度、颜色。
   - 随机生成粒子初始位置与随机颜色。

2. 物理更新
   - 每一帧获取鼠标位置，计算每个粒子到鼠标的方向与距离。
   - 对粒子施加朝向鼠标的引力。
   - 加入空气阻力，使粒子不会无限加速。
   - 粒子碰到窗口边界会自动反弹，不超出画面范围。

3. 渲染阶段
   - main.py 创建窗口，循环执行物理计算与画面绘制。
   - 将 GPU 计算后的粒子位置与颜色绘制到屏幕上。

## 三、实现功能
- 鼠标引力交互：粒子会跟随鼠标移动，距离越近吸引力越强
- 随机柔和配色：从预设的 5 种低饱和度颜色中随机分配
- 物理运动模拟：包含引力、阻力、边界反弹
- GPU 高性能并行计算：支持大量粒子同时流畅运动
- 可自由调节参数：粒子数量、速度、大小、颜色均可在 config.py 中修改
