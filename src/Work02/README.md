# 计算机图形学课程实验二：三维变换与透视投影

项目简介：本项目基于 Python 和 Taichi 实现，完成了三维三角形从空间坐标到屏幕坐标的变换过程。

程序中定义了一个三维空间中的三角形，并依次经过模型变换、视图变换、透视投影、透视除法和屏幕映射，最终在 Taichi GUI 窗口中绘制出来。本实验在三维变换核心流程的基础上，增加了缩放控制、自动旋转、重置操作和不同颜色边框显示。

运行方式：

```bash
uv run python src/Work02/main.py
```

## 二、代码逻辑

1. 初始化阶段

（1）使用 `ti.init(arch=ti.cpu)` 初始化 Taichi 运行环境。

（2）使用 `ti.Vector.field` 定义三维顶点数组 `vertices`，用于保存三角形的三维空间坐标。

（3）使用 `ti.Vector.field` 定义二维屏幕坐标数组 `screen_coords`，用于保存变换后映射到窗口中的二维坐标。

（4）初始化三角形的三个顶点坐标，并创建 Taichi GUI 窗口。

2. 三维变换阶段

（1）构造模型变换矩阵 `Model Matrix`，用于控制三角形自身的旋转和缩放。

（2）构造视图变换矩阵 `View Matrix`，用于模拟相机的位置，将三维物体转换到观察坐标系下。

（3）构造透视投影矩阵 `Projection Matrix`，用于将三维坐标转换到投影空间中。

（4）将三个矩阵组合为 MVP 矩阵：

```text
MVP = Projection × View × Model
```

（5）对三维顶点补充齐次坐标，然后与 MVP 矩阵相乘，得到裁剪空间坐标。

（6）对裁剪空间坐标进行透视除法，将其转换为标准化设备坐标。

（7）将标准化设备坐标从 `[-1, 1]` 映射到 Taichi GUI 使用的 `[0, 1]` 坐标范围。

3. 绘制与交互阶段

（1）使用 `gui.line()` 绘制三角形的三条边。

（2）三条边分别使用不同颜色显示，方便观察三角形的旋转和缩放效果。

（3）通过键盘控制三角形旋转、缩放、自动旋转和重置。

（4）每一帧根据当前角度和缩放比例重新计算三角形顶点位置，并刷新窗口显示结果。

## 三、实现功能

（1）实现三维三角形顶点定义。

（2）实现模型变换，包括绕 z 轴旋转和整体缩放。

（3）实现视图变换，通过相机位置模拟观察坐标系。

（4）实现透视投影，将三维空间坐标投影到二维窗口中。

（5）实现透视除法和屏幕坐标映射。

（6）实现彩色三角形边框绘制，三条边分别使用不同颜色。

（7）实现键盘交互控制。

| 按键 | 功能 |
| --- | --- |
| `A` | 逆时针旋转三角形 |
| `D` | 顺时针旋转三角形 |
| `W` | 放大三角形 |
| `S` | 缩小三角形 |
| `M` | 开启或关闭自动旋转 |
| `R` | 重置角度和缩放比例 |
| `Esc` | 退出程序 |
