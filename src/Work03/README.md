项目简介：本项目基于 Python 和 Taichi 实现，完成了交互式贝塞尔曲线与均匀三次 B 样条曲线的绘制。

程序支持通过鼠标点击添加控制点，并使用 De Casteljau 算法计算贝塞尔曲线上的采样点。同时，程序增加了均匀三次 B 样条曲线绘制功能，可以通过键盘在贝塞尔曲线模式和 B 样条曲线模式之间切换，从而观察两种曲线在几何形态和局部控制特性上的差异。

运行方式：

```bash
uv run python src/Work03/main.py
```

## 二、代码逻辑

1. 初始化阶段

（1）使用 `ti.init(arch=ti.cpu)` 初始化 Taichi 运行环境。

（2）创建像素缓冲区 `pixels`，用于保存窗口中每个像素的颜色。

（3）创建 `gui_points` 和 `gui_indices`，用于在界面中显示控制点和控制多边形。

（4）创建 `curve_points_field`，用于保存 CPU 端计算完成后的曲线采样点，并交给 Taichi kernel 点亮像素。

2. 贝塞尔曲线计算

（1）鼠标左键点击窗口时，将当前鼠标位置加入控制点列表。

（2）当控制点数量不少于 2 个时，程序使用 De Casteljau 算法计算贝塞尔曲线。

（3）De Casteljau 算法的核心思想是递归线性插值。对于给定参数 `t`，先在相邻控制点之间插值得到一组新点，再对新点继续插值，直到最后只剩下一个点，这个点就是曲线在参数 `t` 处的位置。

（4）程序将 `t` 从 0 到 1 进行多次采样，得到整条贝塞尔曲线的离散点。

3. B 样条曲线计算

（1）按下 `B` 键后，程序可以在贝塞尔曲线模式和 B 样条曲线模式之间切换。

（2）B 样条曲线模式下，程序采用均匀三次 B 样条的矩阵形式进行简化计算。

（3）每 4 个相邻控制点构成一段三次 B 样条曲线。

（4）如果共有 `n` 个控制点，则可以生成 `n - 3` 段曲线。

（5）B 样条曲线不会像贝塞尔曲线那样被所有控制点整体影响，而是具有更明显的局部控制特性。

4. 像素绘制与交互

（1）程序先在 CPU 端计算曲线采样点，并将采样点统一传入 `curve_points_field`。

（2）Taichi kernel 根据曲线采样点位置，将像素缓冲区中对应位置点亮。

（3）使用 `canvas.set_image(pixels)` 将像素缓冲区显示到窗口中。

（4）使用 `canvas.circles()` 绘制红色控制点。

（5）使用 `canvas.lines()` 绘制灰色控制多边形，辅助观察控制点之间的几何关系。

## 三、实现功能

（1）实现鼠标点击添加控制点。

（2）实现 De Casteljau 算法绘制贝塞尔曲线。

（3）实现像素缓冲区绘制，通过直接修改像素颜色显示曲线。

（4）实现均匀三次 B 样条曲线绘制。

（5）实现贝塞尔曲线模式与 B 样条曲线模式切换。

（6）实现控制点和控制多边形显示。

（7）实现键盘交互控制。

| 按键 | 功能 |
| --- | --- |
| 鼠标左键 | 添加控制点 |
| `B` | 切换 Bezier 曲线和 B 样条曲线 |
| `C` | 清空所有控制点 |
| `Z` | 撤销上一个控制点 |
| `Esc` | 退出程序 |
