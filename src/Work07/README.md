# 计算机图形学课程实验七：质点-弹簧布料模拟

项目简介：本项目基于 Python 和 Taichi 实现，完成了一个交互式质点-弹簧布料模拟程序。

程序将布料离散为规则网格质点，并通过结构弹簧连接相邻质点。每个质点受到重力、弹簧力和阻尼力的共同作用。实验中实现了显式欧拉、半隐式欧拉和隐式欧拉三种数值积分方法，并通过 GGUI 控制面板进行切换，从而观察不同积分方法在物理模拟稳定性上的差异。

运行方式：

```bash
uv run python src/Work07/main.py
```

## 二、代码逻辑

1. 初始化阶段

（1）使用 Taichi 初始化运行环境，并定义质点位置 `x`、速度 `v`、受力 `f` 等数据场。

（2）将布料离散为 `N × N` 的规则网格，每个网格点作为一个质点。

（3）固定布料上边缘的两个角点，使布料在重力作用下自然下垂。

（4）初始化结构弹簧，连接水平方向和竖直方向相邻的质点。

2. 力的计算

（1）每个质点受到重力作用。

（2）每个质点受到与速度方向相反的阻尼力，用于减少系统能量，防止布料无限震荡。

（3）相邻质点之间通过弹簧连接，弹簧力根据胡克定律计算。

（4）弹簧力由多个弹簧共同作用到同一个质点，因此在并行计算时使用 `ti.atomic_add()` 累加受力。

3. 数值积分

（1）显式欧拉方法先使用旧速度更新位置，再使用当前受力更新速度，计算简单但稳定性较差，容易出现发散现象。

（2）半隐式欧拉方法先更新速度，再用新速度更新位置，相比显式欧拉更加稳定。

（3）隐式欧拉方法使用未来时刻的状态近似计算受力，本实验中采用定点迭代进行近似求解，整体表现更加稳定并带有更明显的阻尼效果。

（4）为了避免数值爆炸，程序加入了速度钳制，当质点速度超过上限时，会将速度限制在最大速度范围内。

4. 渲染与交互

（1）使用 Taichi GGUI 创建三维场景和相机。

（2）使用 `scene.particles()` 绘制蓝色布料质点。

（3）使用 `scene.lines()` 绘制白色弹簧线框，形成布料网格结构。

（4）通过控制面板切换三种积分方法，设置阻尼参数，并控制暂停和重置。

## 三、实现功能

（1）实现质点-弹簧布料建模。

（2）实现结构弹簧连接。

（3）实现重力、弹簧力和阻尼力计算。

（4）实现显式欧拉积分方法。

（5）实现半隐式欧拉积分方法。

（6）实现隐式欧拉近似积分方法。

（7）实现速度钳制，防止数值爆炸。

（8）实现不同阻尼参数对比。

（9）实现 Taichi GGUI 三维渲染和控制面板。

| 操作 | 功能 |
| --- | --- |
| `Explicit Euler` 按钮 | 切换到显式欧拉积分 |
| `Semi-Implicit Euler` 按钮 | 切换到半隐式欧拉积分 |
| `Implicit Euler` 按钮 | 切换到隐式欧拉积分 |
| `Set Damping = 1.0` | 设置阻尼系数为 1.0 |
| `Set Damping = 5.0` | 设置阻尼系数为 5.0 |
| `Pause Simulation` | 暂停模拟 |
| `Resume Simulation` | 继续模拟 |
| `Reset Cloth` | 重置布料状态 |
| 鼠标右键拖动 | 调整观察相机 |
