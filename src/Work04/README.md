项目简介：本项目基于 Python 和 Taichi 实现，完成了一个交互式 Phong 光照模型渲染程序。

程序通过光线与球体、圆锥的相交计算得到物体表面位置和法向量，然后根据 Phong 光照模型计算环境光、漫反射和镜面高光，最终将结果绘制到窗口中。程序提供 UI 滑块，可以实时调节材质参数和光源位置，直观观察不同参数对渲染结果的影响。

运行方式：

```bash
uv run python src/Work04/main.py
```

## 二、代码逻辑

1. 初始化阶段

（1）使用 Taichi 初始化运行环境，并创建像素缓冲区 `pixels`。

（2）定义 Phong 光照模型中的材质参数，包括环境光系数 `Ka`、漫反射系数 `Kd`、镜面反射系数 `Ks` 和高光指数 `shininess`。

（3）定义光源位置参数 `light_x`、`light_y`、`light_z`，用于通过 UI 滑块实时调整光源位置。

（4）创建 Taichi 窗口、画布和 GUI 面板，用于显示渲染结果和交互控件。

2. 几何体相交计算

（1）程序从相机位置向屏幕上每个像素发射一条射线。

（2）使用 `intersect_sphere()` 计算射线与球体是否相交，如果相交，则得到交点距离和球面法向量。

（3）使用 `intersect_cone()` 计算射线与圆锥是否相交，如果相交，则得到交点距离和圆锥表面法向量。

（4）如果同一条射线同时击中多个物体，程序保留距离相机最近的交点。

3. Phong 光照计算

（1）如果射线击中物体，根据交点位置、法向量、光源位置和相机位置计算光照。

（2）环境光 Ambient 用于模拟场景中的基础亮度。

（3）漫反射 Diffuse 根据法向量和光线方向的夹角计算，体现物体表面的受光程度。

（4）镜面高光 Specular 根据反射方向和视线方向的夹角计算，体现光滑表面的高光效果。

（5）最后将环境光、漫反射和镜面高光叠加，并使用 `ti.math.clamp()` 将颜色限制在合法范围内，避免颜色过曝。

4. UI 交互阶段

（1）程序在右上角创建材质和光源参数调节面板。

（2）拖动 `Ka Ambient` 滑块，可以改变环境光强度。

（3）拖动 `Kd Diffuse` 滑块，可以改变漫反射强度。

（4）拖动 `Ks Specular` 滑块，可以改变镜面高光强度。

（5）拖动 `Shininess` 滑块，可以改变高光区域的集中程度。

（6）拖动 `Light X`、`Light Y`、`Light Z` 滑块，可以改变光源位置，从而观察高光位置和明暗变化。

## 三、实现功能

（1）实现基于射线的球体绘制。

（2）实现基于射线的圆锥绘制。

（3）实现物体表面法向量计算。

（4）实现 Phong 光照模型中的环境光 Ambient。

（5）实现 Phong 光照模型中的漫反射 Diffuse。

（6）实现 Phong 光照模型中的镜面高光 Specular。

（7）实现材质参数实时调节。

（8）实现光源位置实时调节。

（9）实现键盘恢复默认参数和退出程序。

| 操作 | 功能 |
| --- | --- |
| 拖动 `Ka Ambient` | 调节环境光强度 |
| 拖动 `Kd Diffuse` | 调节漫反射强度 |
| 拖动 `Ks Specular` | 调节镜面高光强度 |
| 拖动 `Shininess` | 调节高光集中程度 |
| 拖动 `Light X / Y / Z` | 调节光源位置 |
| `R` | 恢复默认参数 |
| `Esc` | 退出程序 |
